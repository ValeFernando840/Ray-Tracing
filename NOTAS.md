## Instalar dependencias desde `requirement.txt`

```
  pip install -r requirements.txt
```
## Para instalar una libreria en particular (example Numpy)
Desde la terminal:
```
  pip install numpy
```

## Para activar el entorno Virtual
```
  venv\Scripts\activate
```
